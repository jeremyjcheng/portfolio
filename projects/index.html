EcoNaut

April 2025 - July 2025

Ocean health monitoring app that detects algal bloom risk and trends

  • Built a full-stack web application with React/TypeScript frontend and Flask backend, featuring ocean health monitoring dashboard with interactive maps and analytics
  • Implemented machine learning pipeline using XGBoost and Facebook Prophet for multivariate time series forecasting, achieving 87% accuracy in algal bloom detection
  • Deployed application using Netlify (frontend) and Render (backend) with PostgreSQL database for scalable data storage and automated CI/CD.

Recipe Rating Predictor

May 2025 - June 2025

Predicts whether a Food.com recipe will be enjoyed using features from ingredients, tags, and contributor history

  • Used the provided Food.com dataset to explore, clean, and prepare training data with scikit-learn and pandas
  • Engineered features including one-hot tags, ingredient flags, rating statistics, contributor encoding, and nutrition ratios
